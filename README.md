# uru-tiny-dots

Pyxel 製の横スクロールランナーゲームです。プレイヤーは自動で走り、ジャンプ・2段ジャンプ・スライディングで障害物や敵を避けます。

## 起動方法

```powershell
pip install pyxel
python main.py
```

## 操作

- `Space` / `Z` / ゲームパッド A: ジャンプ、2段ジャンプ
- 画面右下の `JMP` ボタン: ジャンプ、2段ジャンプ
- `Down` / `S`: スライディング
- ゲームオーバー中の `Space` / `R`: リトライ

## ファイル構成

- `main.py`: 起動用エントリーポイント
- `game.py`: ゲームループ、生成、当たり判定、描画
- `entities.py`: プレイヤー、障害物、崖、背景星のデータ
- `audio.py`: SE と BGM の設定
- `constants.py`: 画面サイズや保存先などの共通定数

## 実装済み要素

- かわいい人型キャラクターと走りアニメーション
- 自動右走りの横スクロールランナー
- 重力、着地リセット、2段ジャンプ
- スライディング
- 踏める敵、踏めない敵、飛行敵、頭上障害物、トゲ床
- ジャンプしすぎると当たる高空敵
- 複数障害物の組み合わせパターンとランダムな間隔変化
- 障害物ごとの個別スクロール速度
- 崖の追加
- 一定時間無敵になるアイテム
- 通常プレイ BGM と無敵中 BGM
- 昼夜変化する背景
- スクロールする地面
- ジャンプ、踏みつけ、ゲームオーバー SE
- ゲームオーバー演出
- コンボスコア
- `save.json` へのハイスコア保存


# PyxelゲームをWeb公開する手順

### ① `.pyxapp` を作成

``` bash
pyxel package . main.py
```

成功すると `.pyxapp` ファイルが生成される

------------------------------------------------------------------------

### ② HTMLに変換（本命）

``` bash
pyxel app2html uru-tiny-dots.pyxapp
```

------------------------------------------------------------------------

## 生成されるファイル

### パターンA（複数ファイル）

    html/
    ├── index.html
    ├── pyxel.js
    └── pyxel.wasm

### パターンB（単一ファイル）

    uru-tiny-dots.html

どちらもブラウザで動作可能

------------------------------------------------------------------------

## 公開方法（GitHub Pages）

### パターンA：htmlフォルダがある場合

``` bash
mkdir docs
cp -r html/* docs/
```

------------------------------------------------------------------------

### パターンB：単一HTMLの場合

``` bash
mkdir docs
mv uru-tiny-dots.html docs/index.html
```

`index.html` にリネームするのが重要

------------------------------------------------------------------------

### Gitへ反映

``` bash
git add .
git commit -m "deploy web version"
git push origin main
```

------------------------------------------------------------------------

## GitHub Pages 設定手順

1.  GitHubリポジトリを開く
2.  「Settings」をクリック
3.  左メニュー「Pages」を選択

------------------------------------------------------------------------

### 公開設定

-   **Source**: Deploy from a branch
-   **Branch**: main
-   **Folder**: `/docs` または `/(root)`

------------------------------------------------------------------------

## ケース別設定

### docsフォルダに配置した場合

    docs/
    └── index.html

-   Branch: main\
-   Folder: `/docs`

------------------------------------------------------------------------

### ルート直下に配置した場合

    index.html

-   Branch: main\
-   Folder: `/(root)`

------------------------------------------------------------------------

## 超重要ポイント

-   ファイル名は必ず `index.html`
-   これが無いと表示されない

------------------------------------------------------------------------

## 公開URL

数十秒〜数分でアクセス可能

    https://ユーザー名.github.io/リポジトリ名/

------------------------------------------------------------------------

## トラブルシューティング

### 404になる

-   `index.html` の配置ミス

### 真っ白

-   キャッシュをクリア（Ctrl + F5）

### 更新されない

-   数分待つ
